# Open-Source Number Guesser 

A lightweight, interactive Python script that challenges users to guess a randomly generated number. This project was developed to demonstrate the core principles of **Free and Open Source Software (FOSS)**, including code transparency, community collaboration, and copyleft licensing.

### Key Features 
* **Dynamic Logic:** Uses the Python `random` module to generate a new challenge every game.
* **Safety First:** Includes `try-except` blocks to handle non-numeric inputs without crashing the program.
* **Lives System:** Limited to 5 attempts to provide a balanced difficulty level.
* **Feedback Loop:** Provides "Higher" or "Lower" hints to guide the user toward the correct answer.

### Installation & Usage 
To run this project locally, follow these steps:

 **Navigate to the Folder:**
1. Ensure you have Python installed.
2. Run `game.py` in your terminal.
